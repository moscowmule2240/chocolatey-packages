No binaries are embedded in this package. At install time it downloads the official
Antigravity IDE installer directly from Google's content-delivery host
(edgedl.me.gvt1.com) - the same host the official download page links to:

    https://antigravity.google/download

The download URL and SHA-256 checksum used for the architecture being installed are
defined in tools\chocolateyinstall.ps1. Chocolatey verifies the checksum during
installation and aborts if it does not match.

tools\chocolateyinstall.ps1 by update.ps1):

To verify a downloaded installer yourself:

    # Windows (PowerShell)
    Get-FileHash -Algorithm SHA256 '.\Antigravity IDE.exe'

    # macOS / Linux
    shasum -a 256 'Antigravity IDE.exe'

Compare the printed hash with the matching checksum above.
